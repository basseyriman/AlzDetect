## 📖 Overview

**AlzDetect** is a web-based application designed to assist in the early detection of Alzheimer's disease through the analysis of brain MRI scans. 
Leveraging advanced machine learning algorithms, the platform provides users with a straightforward interface to upload MRI images and receive analytical insights 
regarding potential anomalies associated with Alzheimer's.

---

## 🎯 Features

- **Vision Transformer (ViT) Analysis**: Leverages state-of-the-art ViT-B/32 architecture for high-resolution MRI classification.
- **Explainable AI (XAI)**: Generates dynamic attention maps to visualize which brain regions influence the model's prediction.
- **Premium SaaS Aesthetic**: A complete redesign featuring glassmorphism, neural gradients, and smooth animations.
- **Four-Stage Classification**: Accurate detection of NonDemented, VeryMild, Mild, and ModerateDemented cognitive stages.
- **Research Portal**: Dedicated sections for methodology, technical documentation, and AI ethics.

---

## 🛠️ Technologies Used

- **AI/ML**: `Vit-Keras`, `TensorFlow`, `OpenAI API` for clinical interpretation.
- **Frontend**: `Next.js 15`, `React`, `Lucide React` (Premium Icons).
- **Backend**: `FastAPI`, `Uvicorn` (Production-grade server).
- **Styling**: `Tailwind CSS` with custom glassmorphism utilities.
- **Deployment**: [Vercel](https://vercel.com/) for frontend, [Uvicorn] for backend hosting.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/) (v14 or later)
- [npm](https://www.npmjs.com/) or [Yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/alzdetect.git
   cd alzdetect
Install dependencies:

bash
npm install
# or
yarn install
Run the development server:

bash
npm run dev
# or
yarn dev
Open in browser: Navigate to http://localhost:3000 to view the application.

🧪 Usage
Upload MRI Scan: Click on the upload area or drag and drop your MRI image (supported formats: SVG, PNG, JPG, GIF).

Preview Scan: Ensure the correct image is uploaded by viewing the preview.

Analyze: Click on "Analyze Scan" to initiate the analysis process.

View Results: Receive insights and potential anomaly detections related to Alzheimer's.
